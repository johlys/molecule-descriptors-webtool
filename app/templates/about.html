About MolDescriptor

MolDescriptor is a simple tool for anyone who wants to know more about molecules. You just enter a molecule using the SMILES notation, and our website tells you about its properties, like its weight or how it might behave in water.

MolDescriptor is still in development, and we need your help to make it better. If you have any suggestions, you can use the form on the feedback page.

RDkit

We are using RDkit to calculate the molecular properties. RDKit is a collection of cheminformatics and machine-learning software written in C++ and Python.

RDkit documentation

You can find the RDkit documentation on their website: https://rdkit.org/docs/index.html

We're Open Source

This means anyone can see how our website works and even make it better. If you know how to code and want to help, you can! Plus, using our website is completely free. Here is our github repository
